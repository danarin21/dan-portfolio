Hindsait

Working closely with the development team lead, we redesigned the user interface for Hindsait's AI-based medical records platform. The data-heavy system needed to be simplified as a digestable interface with the user clearly able to delineate key datasets. I handled all UX/UI design responsibilities.

All names and data are fictional.

Digestable Data

Each page has several essential datasets and tools which users rely on to improve decision-making. We distilled the data to show only the most relevant upon load, with the option to expand results.

Reviewer Dashboard

Whether the user is a physician, care provider, or insurer, relevant data is presented in a concise manner to aid decision-making. Proprietary data mining tools help uncover hidden trends and connections that humans may miss.

Benchmark Analysis

This section contains reviewer data and helps inform the user of behavior patterns/risks. Risk Identifiers illustrate where a reviewer may be biased in their decision-making based on criteria like symptoms, exams, and pairings.

Case Manager

The hub for specific patient cases, the Case Manager houses physician rationales, records, fax history, and activity logs.


The Cohort Manager allows users to save dataset clusters for comparison analysis. As these Cohorts can be recalled or edited at any time, we designed an omni-present click wheel anchored at bottom-left, for quick access.
